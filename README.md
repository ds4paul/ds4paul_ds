# ds4paul_ds

Repozytorium poświęcone projektom i materiałom z zakresu analizy danych (Data Science).

## Zakres tematyczny

- Eksploracyjna analiza danych (EDA)
- Czyszczenie i transformacja danych
- Wizualizacje danych (Matplotlib, Seaborn, Plotly)
- Analiza statystyczna i wnioskowanie
- Praca z danymi tabelarycznymi (Pandas, NumPy)
- Przygotowanie danych do modeli ML
